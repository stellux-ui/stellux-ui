# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x.x   | Yes       |

## Reporting a Problem

If you discover a security-related issue, please report it responsibly.

**Do not open a public GitHub issue for security concerns.**

Instead, send an email to **<luongtruongduong368@gmail.com>** with:

- A description of the issue
- Steps to reproduce (if applicable)
- The version of Stellux UI affected

## Response Timeline

- **Acknowledgement:** within 48 hours
- **Initial assessment:** within 1 week
- **Fix or mitigation:** as soon as reasonably possible

## Disclosure

We will coordinate with you on disclosure timing. We aim to release a fix before any public disclosure.

Thank you for helping keep Stellux UI safe.
